# Rock Paper Scissors

A robust, high-performance vanilla JS implementation of Rock Paper Scissors with advanced state management and animations.

## Features
- **Streak Counter**: Track consecutive wins with glowing indicators.
- **Animated Reveal**: Professional "3-2-1 SHOOT" countdown sequence.
- **Computer Personalities**: Three distinct weighted CPU behaviors.
- **Move History**: Visual log of recent rounds.
- **Light Theme**: Stark editorial design using Space Grotesk.

## Instructions
1. Open `index.html` in any modern web browser.
2. First to 3 wins takes the operation.
